# project_multimedia
project memory puzzle: oplossen van memory game aan de hand van videobeelden

## Requirements

- Python: 3.6.x
- requirements.txt

## Taakverdeling
- Bjarne: Array maken van goed uitleesbare frames + controle matching
- Rob: Uitsnijden tegels + matching

## Run script
1. Plaats de video's in een map 'Memory_Games' in de root van het project
2. Pas main.py aan met video uit onderstaande lijst en run het script

Om alle video's te testen, voer SearchKeyFramesSelected.py uit
